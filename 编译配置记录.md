# Subnet Planner 编译配置记录

## 1. 编译配置概述

当前编译配置使用 Nuitka 作为主要编译器，生成的可执行文件大小约为 17MB，压缩比达到 26.11%。最重要的是，**360等杀毒软件不再报毒**。

## 2. 核心编译命令

```bash
python -m nuitka --onefile \
--windows-icon-from-ico=Subnet_Planner.ico \
--include-data-file=translations.json=translations.json \
--include-data-file=Subnet_Planner.ico=Subnet_Planner.ico \
--include-data-file=icon.ico=icon.ico \
--include-data-dir=Picture=Picture \
--enable-plugin=tk-inter \
--windows-console-mode=disable \
--product-name="Subnet Planner" \
--product-version=2.7.0 \
--file-version=2.7.0 \
--company-name="Subnet Planner Team" \
--copyright="Copyright © 2025-2026 Subnet Planner Team" \
--file-description="子网规划师 - IP子网规划工具" \
--output-filename=SubnetPlannerV2.7.0.exe \
windows_app.py
```

## 3. 关键优化点（可能是不再报毒的原因）

### 3.1 移除了危险的编译选项
- 移除了 `--noinclude-unittest`、`--nofollow-imports` 等可能引起杀毒软件怀疑的选项
- 移除了 `--remove-output` 选项，避免了杀毒软件对临时文件的误判

### 3.2 启用了压缩功能
- Nuitka 自动启用了压缩功能，压缩比达到 26.11%
- 压缩后的文件结构更紧凑，减少了杀毒软件的误判概率

### 3.3 完整的版本信息
- 提供了完整的公司名称、产品名称、版权信息、文件描述等
- 这些信息使可执行文件看起来更正规，减少了杀毒软件的怀疑

### 3.4 优化了依赖管理
- 仅包含必要的依赖文件
- 避免了包含可能引起杀毒软件怀疑的不必要模块

### 3.5 代码签名（可选）
- 支持使用 PFX 证书对可执行文件进行签名
- 签名后的文件更可信，减少了杀毒软件的误判

## 4. 编译脚本使用方法

### 4.1 使用 build_compile.py（推荐）
```bash
# 默认使用 Nuitka 单文件模式编译
python build_compile.py

# 显示帮助信息
python build_compile.py --help
```

### 4.2 使用 build_compile.bat
```batch
# 默认使用 Nuitka 单文件模式编译
build_compile.bat

# 显示帮助信息
build_compile.bat --help
```

### 4.3 使用 build_compile.ps1
```powershell
# 默认使用 Nuitka 单文件模式编译
.\build_compile.ps1

# 显示帮助信息
.\build_compile.ps1 --help
```

## 5. 版本管理机制

- 所有版本信息集中在 `version.py` 中管理
- 编译前自动生成 `version_info.py`，确保版本信息一致
- 生成的可执行文件名包含完整版本号（如 `SubnetPlannerV2.7.0.exe`）

## 6. 注意事项

1. **杀毒软件设置**：编译过程中可能需要暂时关闭或排除编译目录的杀毒软件实时监控
2. **临时文件**：当前配置保留了临时文件目录，便于调试和分析
3. **依赖更新**：定期更新 Nuitka 和其他依赖，以获取最新的优化和安全修复
4. **代码签名**：建议使用有效的代码签名证书，进一步减少杀毒软件误报
