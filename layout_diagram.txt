子网规划面板布局图
==================

整个窗口布局：
┌──────────────────────────────────────────────────────────────────────┐
│                     子网规划页面                        │
├──────────────────────────────────────────────────────────────────────┤
│  Row 0 (weight=0) - 固定高度                           │
│  ┌──────────────┐  ┌────────────────────────────────────┐   │
│  │  父网段设置  │  │        子网需求 (width=250)      │   │
│  │  (width=250) │  │        [子网需求表格]             │   │
│  │              │  │        [操作按钮]                 │   │
│  └──────────────┘  │                                 │   │
│                 │  └────────────────────────────────────┘   │
├──────────────────────────────────────────────────────────────────────┤
│  Row 1 (weight=1) - 不可伸缩高度                           │
│  ┌──────────────┐                                       │
│  │  需求池     │                                       │
│  │  (width=250) │                                       │
│  │  [需求池表格]  │                                       │
│  │  [滚动条]     │                                       │
│  └──────────────┘                                       │
├──────────────────────────────────────────────────────────────────────┤
│  Row 2 (weight=1) - 可伸缩高度                           │
│  ┌──────────────────────────────────────────────────────────────┐   │
│  │                规划结果                    │   │
│  │  [规划结果表格]                                 │   │
│  └──────────────────────────────────────────────────────────────┘   │
└──────────────────────────────────────────────────────────────────────┘

Grid配置：
- 列配置：
  - Column 0 (weight=1): 左侧列可伸缩，包含父网段设置和需求池
  - Column 1 (weight=1): 右侧列可伸缩，包含子网需求

- 行配置：
  - Row 0 (weight=0): 父网段设置，固定高度
  - Row 1 (weight=0): 需求池和子网需求，固定高度
  - Row 2 (weight=1): 规划结果，可伸缩高度

面板位置：
- 父网段设置：grid(row=0, column=0)
- 子网需求：grid(row=0, column=1, rowspan=2)
- 需求池：grid(row=1, column=0)
- 规划结果：grid(row=2, column=0, columnspan=2)

面板外边距详情：
1. 父网段设置面板：
   - 内边距：padding=(10, 10, 0, 10) (上10，右10，下0，左10)，右边距为0
   - 外边距：padx=(0, 5), pady=(0, 10) (左0，右5，上0，下10)
   - 位置：grid(row=0, column=0, sticky="ew")

2. 需求池面板：
   - 内边距：padding=(10, 10, 0, 10) (上10，右10，下0，左10)
   - 外边距：padx=(0, 5), pady=(0, 10) (左0，右5，上0，下10)
   - 位置：grid(row=1, column=0, sticky="nsew")

3. 子网需求面板：
   - 内边距：padding=(10, 10, 0, 10) (上10，右10，下0，左10)
   - 外边距：padx=(5, 0), pady=(0, 10) (左5，右0，上0，下10)
   - 位置：grid(row=0, column=1, rowspan=2, sticky="nsew")

4. 规划结果面板：
   - 内边距：padding="10" (上下左右均为10像素)
   - 外边距：pady=(0, 0) (上下均为0)
   - 位置：grid(row=2, column=0, columnspan=2, sticky="nwse")

特点：
1. 左侧列固定宽度，右侧列可伸缩
2. 父网段设置在左上角
3. 需求池在左下角
4. 子网需求面板在右侧，跨两行
5. 规划结果在底部，跨两列
6. 滚动条始终显示，不再自动隐藏
7. 表格列宽可自适应调整
