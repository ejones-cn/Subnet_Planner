# IP子网切分工具运行计划

## 项目概述
IP子网切分工具是一个功能强大的网络工具，支持CIDR输入、子网切分、剩余网段自动计算等功能，提供Web界面和Windows GUI界面两种使用方式。

## 运行前准备
1. **检查Python环境**：确保已安装Python 3.x
2. **安装依赖**：运行 `pip install -r requirements.txt` 安装项目依赖

## 运行方式选择
### 选项1：运行Web界面
- 命令：`python web_app.py` 或 `.run_web_app.bat`
- 访问地址：http://localhost:5000
- 特点：跨平台支持，通过浏览器访问，适合各种操作系统

### 选项2：运行Windows GUI界面
- 命令：`python windows_app.py` 或 `.run_app.bat`
- 特点：本地桌面应用，支持结果导出功能，适合Windows用户

### 选项3：直接运行可执行文件
- 路径：`dist/IP子网分割工具.exe`
- 特点：无需Python环境，直接双击运行，适合没有Python环境的Windows用户

## 推荐运行方式
由于用户没有指定具体运行哪个界面，我将默认运行**Windows GUI界面**，因为这是项目的主要界面，且功能更完整（支持结果导出）。

## 执行步骤
1. 检查并安装项目依赖
2. 运行Windows GUI界面
3. 验证应用是否正常启动

## 预期结果
- 应用成功启动，显示主界面
- 可以正常输入CIDR地址进行子网切分
- 计算结果正确显示

## 故障排除
- 如果依赖安装失败，检查Python版本和pip配置
- 如果应用启动失败，查看控制台输出的错误信息
- 如果Web界面无法访问，检查端口是否被占用

请确认是否执行此运行计划，或选择其他运行方式。
