# 移动导出结果按钮到切分结果面板

## 目标

将当前位于输入参数面板的"导出结果"按钮移动到"切分结果"面板中，风格参考子网规划模块的"导出规划"按钮。

## 实现步骤

1. **移除原按钮**：删除输入参数面板中的导出结果按钮

   * 位置：windows\_app.py 第1105-1106行

   * 代码：`self.export_btn = ttk.Button(input_frame, text="导出结果", command=self.export_result, width=8)`

   * 布局：`self.export_btn.grid(row=0, column=4, rowspan=2, padx=(8, 3), pady=3, sticky=tk.N + tk.S + tk.E + tk.W)`

2. **在切分结果面板添加新按钮**：在create\_split\_result\_section方法中添加导出结果按钮

   * 位置：windows\_app.py 第1221-1227行之间，result\_frame创建后

   * 使用与导出规划按钮相同的样式和布局

   * 样式：Accent.TButton（蓝色系，白色文字，加粗字体）

   * 布局：place布局，右上角，距离右边0像素，距离顶部-3像素

3. **应用相同样式**：确保按钮使用与导出规划按钮相同的样式

   * 样式定义：windows\_app.py 第1521-1529行的Accent.TButton样式

   * 已在应用初始化时创建，可直接使用

4. **保持功能不变**：确保新按钮仍调用原export\_result方法

## 预期效果

* 导出结果按钮将显示在"切分结果"面板的右上角，与"导出规划"按钮风格一致

* 按钮颜色：蓝色系

* 按钮文字：白色、加粗

* 位置：右上角，与面板边缘对齐

* 功能：保持原有的导出结果功能

## 代码修改点

1. 删除 windows\_app.py 第1105-1106行的导出按钮代码
2. 在 windows\_app.py 第1224行（result\_frame.pack后）添加新的导出按钮代码

## 示例代码

```python
# 在create_split_result_section方法中添加
# 导出结果按钮 - 使用 place 布局手动控制位置
export_btn = ttk.Button(result_frame, text="导出结果", 
                        command=self.export_result, 
                        width=10,
                        style="Accent.TButton")
# 手动指定按钮位置：右上角，距离右边0像素，距离顶部-3像素
export_btn.place(relx=1.0, rely=0.0, anchor=tk.NE, x=0, y=-3)
```

## 技术细节

* 使用place布局确保按钮始终位于右上角
* 使用与子网规划模块相同的按钮样式（Accent.TButton）
* 保持原有的导出功能不变
* 按钮大小统一为10，与导出规划按钮一致
